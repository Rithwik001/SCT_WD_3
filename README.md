# 🎮 Tic-Tac-Toe Web Application

## 📌 Project Overview

This project is a simple and interactive **Tic-Tac-Toe** game developed using **HTML, CSS, and JavaScript**. It is a two-player game where players take turns placing **X** and **O** on a 3×3 grid. The game automatically checks for a winner or a draw after every move and provides an option to restart the game.

---

## ✨ Features

* 🎲 Two-player gameplay (Player X vs Player O)
* ✅ Automatic winner detection
* 🤝 Draw detection
* 🔄 Restart game functionality
* 📱 Responsive and user-friendly interface
* ⚡ Fast and interactive gameplay
* 🎨 Clean and simple design

## 🎮 How to Play

1. Player **X** always starts the game.
2. Players take turns clicking an empty cell.
3. The first player to place **three identical symbols** in a row, column, or diagonal wins.
4. If all nine cells are filled without a winner, the game ends in a **Draw**.
5. Click the **Restart Game** button to start a new match.

---

## 📸 Game Functions

* Tracks player turns
* Prevents selecting an occupied cell
* Checks all winning combinations
* Detects draw conditions
* Displays the game result
* Resets the board with one click
